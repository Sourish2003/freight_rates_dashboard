## Getting Started

This project is a starting point for a Flutter application.

1. Clone the repository

2. Install dependencies:

		cd freight_rates_dashboard flutter pub get

	or if using integrated terminal, use:
    
    	flutter pub get

3. Run the app:

		flutter run
